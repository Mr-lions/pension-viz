# 农村养老金强互动可视化｜完整文案政策版

## 打开方式
直接双击 `index.html`，建议使用最新版 Chrome 或 Edge。项目已内置 Plotly，不需要联网安装依赖。

## 主要文件
- `index.html`：页面结构与完整正文
- `styles.css`：页面视觉与响应式样式
- `data.js`：图表数据、完整政策时间线、地方案例和国外政策文本
- `app.js`：图表与交互逻辑
- `assets/`：图表库和案例图片
- `source_materials/`：本次使用的纯文案与政策数据表
